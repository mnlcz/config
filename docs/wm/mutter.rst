Mutter (GNOME)
==============

This section documents keybinding configuration for GNOME.

Overview
--------

These files define GNOME keybindings and are used as a backup or alternative
environment.

Configuration Files
-------------------

- ``media-keys.conf``
- ``shell-keybindings.conf``
- ``wm-keybindings.conf``

These files are used with scripts:

- :doc:`../scripts/index`

  - ``gnome-save-keybindings.sh``
  - ``gnome-restore-keybindings.sh``

Usage
-----

- Save current keybindings
- Restore predefined configuration

Notes
-----

- Intended for GNOME environments only
- Separate from the main Wayland (dwl) setup
