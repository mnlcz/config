Dotfiles & Configuration
========================

This repository contains my personal configurations, scripts, and templates 
for a Linux-based development environment.

It includes:

- Editor configuration
- Shell environment
- Window manager configuration
- Utility scripts
- Supporting tools
- Reusable templates

Structure
---------

The repository is organized into the following directories:

- ``editor/``: editor configurations (Neovim, Acme, etc.)
- ``shells/``: shell environment and initialization
- ``wm/``: window manager configuration
- ``scripts/``: automation and utilities
- ``tools/``: supporting tools (tmux, fastfetch)
- ``term/``: terminal configuration
- ``mpv/``: media player configuration
- ``templates/``: reusable project templates
